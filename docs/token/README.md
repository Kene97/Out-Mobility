## Status

**$OUT is not yet launched.**

All tokenomics figures, allocations, and mechanics described in this documentation
are **proposed** and subject to change prior to the Token Generation Event (TGE).

This documentation represents the current thinking and design intent of the
Out Mobility team. It will be updated as the product matures.

---

## Positioning Statement

$OUT is not a speculative asset.

It is the operational currency of a physical mobility network — a token designed
to flow through real transactions, reward real contributors, and govern real
infrastructure as Out Mobility scales.

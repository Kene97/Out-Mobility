# Operating Rhythm

How Out Mobility operates week to week.

---

## Weekly Cadence

### Monday — Week Set
- Review open milestones
- Identify the 3 most important things to advance this week
- Block time for each
- No meetings on Monday mornings

### Wednesday — Mid-Week Check
- Are the 3 priorities moving?
- Any blockers to clear?
- Any decisions that need to be made?

### Friday — Week Close
- What shipped?
- What is stuck and why?
- What do we carry into next week?
- Update milestone tracker

---

## Decision-Making Framework

**Two types of decisions:**

| Type | Description | How to decide |
|---|---|---|
| Reversible | Easy to undo, low cost of error | Decide fast, move, adjust |
| Irreversible | Hard to undo, high cost of error | Slow down, think clearly, document |

**Default posture:** Move fast on reversible decisions. Never confuse speed with recklessness on irreversible ones.

**Escalation rule:** If a decision costs >$1,000 or affects an external party (operator, advertiser, investor), flag it before acting.

## Metrics We Track

### Weekly
- Vehicles with active tablets (count)
- Active advertiser campaigns (count)
- Impressions delivered (count)
- Open operator conversations (count)
- Open investor conversations (count)

### Monthly
- MRR (monthly recurring revenue)
- Revenue per vehicle
- CPM achieved
- Operator churn / satisfaction
- Advertiser renewal rate

---

## Communication Principles

- Decisions are documented, not just discussed
- If it matters, it is written down
- Short updates > long meetings
- Async first, sync when necessary

---

## Prioritization Rule

If a task does not directly:
- deploy more vehicles,
- sign more advertisers,
- improve impression verification, or
- support fundraising —

it is not a priority this week.

---

## Fundraising Rhythm

While not fundraising full-time, maintain a steady investor pipeline:
- 2–3 new investor introductions per month
- Monthly update email to warm investors
- Every milestone achieved = investor message sent within 48 hours
